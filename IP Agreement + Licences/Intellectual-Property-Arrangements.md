# Intellectual Property Arrangements

## Foreground IP
The project uses the existing **Discourse Forum Platform which is licenced under GPLv2**. Since we are only configuring Discourse and not modifying its source code, background IP will **remain owned by its original authors**. The following artifacts constitute the foreground intellectual property created within the project:
- Configuration tailored to Next Chapter Scotland's requirements of the Discourse instance
- Next Chapter Scotland's Theme created by using their Branding Assets
- Deployment and hosting setup for Next Chapter Scotland
- Project Documentation, such as configuration guides and deployment instructions
- General, non-NCS (Next Chapter Scotland) specific modifications to Discourse instance
- The App developed for Next Chapter Scotland, including source code written by Team SH39

In the event that **standalone forum software** is developed, it will fall under the **GPLv3 License**.

## Ownership of IP
Ownership of this project is assigned as follows:  
#### Next Chapter Scotland:
- All branding assets
- NCS Branding Theme
- Their deployed instance of the Forum
- NCS specific UI content and customsisation (if any)
- Branding associated with the App

#### Team SH39:
- Any contribution/modifications that are not for NCS
- All general-purpose documentation 
- Any tooling, scripts, or reusable deployment materials not containing NCS branding
- Any standalone forum software developed independently by the team
- The source code of the App (excluding NCS branding and content)

#### **Team SH39** does **not** own:
- Discourse software (GPLv2-or-later)
- NCS branding
- The final deployed instance

## Compensation and Transfer Conditions
No financial compensation is exchanged as part of the IP transfer.
